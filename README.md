## Requirements

- iOS 9.0+
- Xcode 14

## Installation

#### CocoaPods
Install the pod libraries using [CocoaPods](http://cocoapods.org/):

```ruby
pod install
```

## Usage

```swift
cd directory
open .countdown.xcworkspace
```
